# Icônes Polytopia

Ce dossier contient les icônes utilisées pour visualiser le jeu Polytopia.

## Structure

- `terrain/` : Icônes pour les types de terrain (plaine, forêt, montagne, eau)
- `units/` : Icônes pour les types d'unités (guerrier, archer, défenseur, etc.)
- `cities/` : Icônes pour les villes et châteaux
- `effects/` : Icônes pour les effets visuels (soin, attaques, etc.)
- `tribes/` : Icônes pour les tribus
- `tech/` : Icônes pour les technologies

## Icônes disponibles pour Imperius et Xin-xi

### Unités

Les icônes d'unités suivent la convention de nommage : `[Type][Tribu].png`
- **I** = Imperius
- **X** = Xin-xi

#### Imperius (I)
- `units/WarriorI.png` - Guerrier
- `units/ArcherI.png` - Archer
- `units/DefenderI.png` - Défenseur
- `units/KnightI.png` - Chevalier
- `units/GiantI.png` - Géant
- `units/ShipI.png` - Navire
- `units/BattleshipI.png` - Cuirassé

#### Xin-xi (X)

## Notes d'utilisation

- Toutes les icônes sont au format PNG (pas de SVG disponible actuellement)
- Les icônes d'unités sont spécifiques à chaque tribu (style visuel différent)
- Les terrains Imperius peuvent être utilisés comme base pour d'autres tribus classiques
- Les icônes de villes peuvent être utilisées selon le niveau de la ville (château pour les villes de niveau élevé)
